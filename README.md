# duckweed-rmbe

This repository contains the experimental data, custom image analysis software, and statistical scripts evaluating the growth dynamics and photoinhibition of _Lemna minor_ under different supplemental LED light intensities. This research was done at WUR for course FTE25806 Research Methods Biosystems Engineering.

## Project Structure

- `data/`: Contains raw CSV data files for biomass and surface coverage.
- `src/image_analysis/`: Script to process top-down images and calculate 2D surface coverage percentages.
- `src/statistics/`: R scripts for data processing, plotting, and running statistical tests.
- `output/`: Generated figures.
